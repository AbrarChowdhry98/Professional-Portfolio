### Description

Personal portfolio for Abrar Chowdhry, built with Next.js, Emotion, and TypeScript. Design inspired by [portfolio-v2](https://github.com/iqtedar123/portfolio-v2). Content is managed locally — no CMS required.

### Setup

1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000)

### Build

`npm run build`

### Customize

Edit `lib/contentful.ts` to update banner text, projects, social links, and contact information.

### Deploy

The project can be deployed to Vercel, Firebase Hosting, or any Next.js-compatible platform:

`npm run build && npm run start`
